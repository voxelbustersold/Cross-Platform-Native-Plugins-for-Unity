Address Book

This feature allows you to access contact information from the Address Book.

Auth Status

Check if Address Book access is authorised

Read Contacts

Get list of users contacts with details such as Name, Picture, Email, etc.

Editor Simulation

Simulate Address Book behaviour on Editor to ease development process 

Initialisation

Follow below steps in order to use this feature

1. Check Authorization

2. Read Contacts

Next

Check Authorization Status

This method will let you know if you have the permission to access Address Book.

Make sure you call this method to check for access status and then start on reading contacts. 

On iOS platform, this call will show an Alert to let user to give access. 

NPBinding.AddressBook.GetAuthorizationStatus();

Code Sample

Read Contacts

NPBinding.AddressBook.ReadContacts(OnReceivingContacts);

This takes a callback which will get triggered once reading contacts is finished.

Callback returns an array of AddressBookContact instances and authorization status.

Implementation Example

               
using UnityEngine;
using System.Collections;
using VoxelBusters.NativePlugins;
public class ExampleClass : MonoBehaviour 
{      
    public bool GetAuthorisationStatus()
    {
       	bool _status = NPBinding.AddressBook.GetAuthorizationStatus();
        return _status;
    }
    public void ReadContacts ()
    {
        NPBinding.AddressBook.ReadContacts(OnReadContactsFinished); 
    }
    private void OnReadContactsFinished (eABAuthorizationStatus _authorizationStatus, AddressBookContact[] _contactList)
    {
        if (_authorizationStatus == eABAuthorizationStatus.AUTHORIZED)
        {
            // Insert your code to handle contact info
        }
        else
        {
            // Something went wrong
        }
    }
}

Editor Simulation

Address Book editor simulation provides optional functionality to test your code on Unity Editor.

Access Address Book Simulation from
Window- >Voxel Busters -> Native Plugins - > Editor Simulation - > Select Address Book.

Tip : Try setting different Authorization Status and make sure your code handles all possible invalid cases when deployed on target device.

FAQ

Where I can use this feature?

If you need to access address book of your player to get contacts.

What details I get from each contact?

First Name, Last Name, Phone Number List, Email ID List, Image (Which can be accessed with GetImageAsync method).
