Property Documentation

string ContentTitle
getset

The first line of text in the notification.

This is the text that is displayed in the notification bar on Android as title of the notification.

string TickerText
getset

The ticker text.

The text that will be visible in a scrolling fashion on status bar.

Note
All the devices using pre-Lollipop OS has this feature.
string Tag
getset

The tag of the notification.

The tag defines this notification uniquely or can be empty which overwrites previous notification. If the tag is set with different value than previous notification, it won't override the previous one in notification bar, otherwise it will.

string CustomSound
getset

The name of the sound file to play when an alert is displayed.

Note
Make sure all referred sounds are in Assets/PluginResources/Android or Common folder.
string LargeIcon
getset

The image used as the large icon for notification.

Note
This will be the icon thats displayed in the notification. If the value is not set, then default image will be used.
